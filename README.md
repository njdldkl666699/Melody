

# 写在前面

#### 编码规范

* 窗口大小模定为``QSize(1280,720)``
* 类名使用大驼峰命名法``SettingsWidget``，而函数名和变量名使用小驼峰命名法``playWidget``
* 组件/控件对象  组件控件_名字  ``label_musicVolume``
* 中文字体使用思源黑体，西文字体使用Saira

#### 文件组织

* ./src/\*.cpp
* ./include/\*.h
* ./ui/\*.ui
* ./res/{\$资源文件}
