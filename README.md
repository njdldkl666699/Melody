# 编码规范

* 窗口大小模定为``QSize(1440,819)``
* 类名使用大驼峰命名法``SettingsWidget``，而函数名和变量名使用小驼峰命名法``playWidget``
* 组件/控件对象  组件控件_名字  ``label_musicVlume``
* 字体使用华文中宋，font 15 粗体


# 文件组织

* src文件存放所有代码
* res文件存放资源
* ui文件存放.ui文件
* include文件存放所有头文件
